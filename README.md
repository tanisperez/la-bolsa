# La Bolsa

## Detalles técnicos
El proyecto está construido utilizando las siguientes tecnologías:
* Node.js v16.15.1
* React 18.2.0
* Next.js 12.2.0

## Ejecución del servidor de desarrollo
```bash
npm run dev
# or
yarn dev
```

## Construcción del proyecto y arranque
```bash
npm run build
npm run start
```
